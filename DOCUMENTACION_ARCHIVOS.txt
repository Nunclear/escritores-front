CÓMO USAR:
═════════════════════════════════════════════════════════════════════════════

1. INSTALAR MKDOCS:
   pip install mkdocs mkdocs-material pymdown-extensions

2. SERVIR LOCALMENTE:
   mkdocs serve
   → Ir a http://localhost:8000

3. GENERAR HTML:
   mkdocs build
   → Genera carpeta 'site/'

4. DESPLEGAR EN GITHUB PAGES:
   mkdocs gh-deploy
