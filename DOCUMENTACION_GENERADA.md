## 🚀 Cómo Usar

### Opción 1: Servidor Local (Desarrollo)

```bash
# Instalar MkDocs
pip install mkdocs mkdocs-material mermaid2

# Ejecutar servidor
cd /vercel/share/v0-project
mkdocs serve

# Acceder a
http://localhost:8000
```

### Opción 2: Build Estático

```bash
# Build HTML
mkdocs build

# Archivos generados en carpeta 'site/'
# Subir a GitHub Pages, Vercel, Netlify, etc.
```

### Opción 3: Publicar en Vercel/Netlify

1. **Vercel**:
   - Push a GitHub
   - Ir a vercel.com
   - Importar repo
   - Build command: `pip install mkdocs mkdocs-material mermaid2 && mkdocs build`
   - Output directory: `site`

2. **Netlify**:
   - Similar a Vercel
   - Deploy en `site/` folder

3. **GitHub Pages**:
   - `mkdocs gh-deploy`
   - Automático en `gh-pages` branch
